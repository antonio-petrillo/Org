* What is it used for
Show the activity and the control flow in a process, it is an updated version of [[https://en.wikipedia.org/wiki/Flowchart][Flowchart Diagram]].
* Summary
  | Symbol           | Description               |
  |------------------+---------------------------|
  | ●                | Start of the diagram      |
  | ⦿                | End of the diagram        |
  | rectangle + text | activity with description |
  | black pipe       | fork/join                 |
  | rhombus          | alternative (if/else)     |
