* Incremental
Basically the steps are the same of the /Cascade/ model but in the /Incremental/ approach there are a few key difference.
Instead of develop the entire system in one shot it identifies the most important features to implement first, after that the software is already delivered to the clients.
If the clients accepts/like the implementation the model continue to implement the remaining feature producing various release, each one with more functionality, hence incremental release.
One of the most famous incremtal process is [[denote:20230930T115110][Scrum]].
